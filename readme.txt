Crea fácilmente una página de contactos del personal a partir de un CSV o Google Workspace. Compatible con Beaver Builder para personalizar la interfaz.
== Description ==
Este plugin permite mostrar un directorio de empleados en tu sitio WordPress usando datos de Google Workspace o desde un archivo CSV público.
Ideal para crear páginas de contacto internas o directorios de personal con campos como nombre, correo electrónico, puesto, departamento y extensión telefónica.
